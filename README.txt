Exercício realizado na matéria de programação WEB do 4º Período do curso de G.T.i - FAPAM

Proposta:
Criar um formulário com o campo nome de Usuário que ao perder o foco deve verificar se tem no mínimo 5 caracteres, caso não tenha, deverá aparecer uma mensagem do lado do campo informando o problema. Caso esteja ok, o campo deverá ficar verde.

Também deverá ter dois campos senha e repetir senha. Quando o usuário clicar em enviar, deverá ser verificar se as senhas são iguais, caso não sejam, informar ao usuário.

Ao decorrer do trabalho realizei algumas alterações para ficar mais intuitivo e desenvolvi um layout para ficar mais agradável.
